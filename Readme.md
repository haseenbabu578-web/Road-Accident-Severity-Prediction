🚗 Road Accident Severity Prediction

This project predicts whether a road accident will be Severe or Not Severe using a Machine Learning model. The application is built with Python, Scikit-learn, and Streamlit to provide an interactive web interface where users can input accident-related details and get predictions instantly.

📌 Project Overview

Road accidents cause serious damage to life and property. Predicting accident severity can help authorities and emergency services respond faster and improve road safety strategies.

This project uses a trained machine learning model to predict accident severity based on several factors such as:

State

City

Vehicle Type

Cause of Accident

Number of Injuries

Weather Conditions

Road Type

Month

Hour of Accident

The model outputs whether the accident is:

⚠️ Severe Accident

✅ Not Severe Accident
